Introduccion

JavaScript es un lenguaje de programación ligero e interpretado.

Es conocido como el lenguaje de las páginas web.

console.log("Hola Mundo");
  • Multiplataforma
  • Orientado a objetos
Variables

Utilizamos variables como contenedores de valores.

Puedes declarar una variable usando var, let o const.

let nombre = "Gemini";
  • Let es para bloques
  • Const es para constantes
Funciones

Las funciones son bloques de código reutilizables.

Se definen con la palabra clave function.

function saludar() { return "Hola"; }
Objetos

Los objetos permiten agrupar datos relacionados.

Se definen con llaves {}.

const coche = { marca: "Tesla" };
Arrays

Los arrays son listas de elementos ordenados.

El primer elemento siempre está en el índice 0.

let lista = [1, 2, 3];
  • Método push()
